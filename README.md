# Audio Noise Inverter System (Capstone Project)

## Overview
Multidisciplinary systems engineering project focused on developing a real-time audio-based noise inverter system. The goal was to reduce a user’s auditory footprint by capturing environmental sound and generating an inverted output signal.

## My Role
- Led systems engineering planning for a cross-functional team  
- Defined project scope, deliverables, and timelines  
- Coordinated between engineering sub-teams to ensure alignment and progress  

## Key Contributions
- Developed a Work Breakdown Structure (WBS) and WBS Dictionary to define system scope and deliverables  
- Applied PERT/CPM scheduling to manage project timelines and dependencies  
- Conducted Failure Modes and Effects Analysis (FMEA) to identify and mitigate critical risks  
- Supported system-level design decisions involving real-time audio processing and hardware integration  

## System Concept
The system captures environmental audio through a microphone and generates an inverted output signal through speakers to reduce detectable sound. The design integrates sensing, processing, and output components to operate in real time.
